# 開放電腦計畫 -- 計算機硬體結構 (使用 Verilog 實作)

## 關於本書

本書是「開放電腦計畫的硬體部份」，描述如何設計電腦硬體的方法，透過這本書，我們希望讓「計算機結構」這門課程，變成是可以動手實作的。

我們相信，透過實作的訓練，您將對理論會有更深刻的體會，而這些體會，將會進一步讓您更瞭解「現代電腦工業的結構」是如何建構出來的。
